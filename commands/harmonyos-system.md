# HarmonyOS Command Prompt: /harmonyos-system

Use this prompt to run the `harmonyos-deveco` design-system checkpoint.

## Arguments

`$ARGUMENTS`

## Delegation

Apply the `harmonyos-deveco` skill.

- Run only `/harmonyos-system` unless the user explicitly asks to continue.
- Produce an ArkUI-ready design system: semantic color, type, spacing, radius, elevation, motion, accessibility, dark mode, and component roles.
- Read `references/arkui-design-system.md` when design tokens, responsive behavior, or component ownership are unclear.
- Do not write ArkTS or edit DevEco files in this checkpoint.
