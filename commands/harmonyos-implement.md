# HarmonyOS Command Prompt: /harmonyos-implement

Use this prompt to run the `harmonyos-deveco` implementation checkpoint.

## Arguments

`$ARGUMENTS`

## Delegation

Apply the `harmonyos-deveco` skill.

- Run `/harmonyos-implement` only after there is enough Product Ideal, UX, design-system, and ArkUI ownership context to avoid generic page-building.
- Inspect the current DevEco project before editing: `oh-package.json5`, `build-profile.json5`, `hvigorfile.ts`, `AppScope/app.json5`, `entry/src/main/module.json5`, `entry/src/main/ets`, and `entry/src/main/resources`.
- Make focused ArkTS/ArkUI changes that follow local project conventions.
- Keep routes, resources, permissions, build, and signing impact explicit.
