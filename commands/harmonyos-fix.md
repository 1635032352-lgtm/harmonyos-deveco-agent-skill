# HarmonyOS Command Prompt: /harmonyos-fix

Use this prompt to run the `harmonyos-deveco` debugging checkpoint.

## Arguments

`$ARGUMENTS`

## Delegation

Apply the `harmonyos-deveco` skill.

- Capture the exact command/error and the first project-owned failure line.
- Search the exact error token plus one context token: API name, config key, module name, permission, route, resource, or hvigor task.
- Produce a root-cause note, matched docs/search terms, minimal patch plan, rerun command, and remaining blockers.
- Patch only the narrow cause, then rerun the same command when practical.
