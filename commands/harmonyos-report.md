# HarmonyOS Command Prompt: /harmonyos-report

Use this prompt to run the `harmonyos-deveco` handoff-report checkpoint.

## Arguments

`$ARGUMENTS`

## Delegation

Apply the `harmonyos-deveco` skill.

- Produce a concise completion report tying product ideal, UX, ArkUI ownership, changed files, validation evidence, and remaining risks together.
- Preserve evidence honesty: static review, build success, unsigned HAP, signed HAP, Preview, and device proof are different evidence levels.
- Mark unresolved environment or device gaps as CONDITIONAL PASS or BLOCKED.
- Do not invent commands, artifacts, screenshots, or timestamps.
