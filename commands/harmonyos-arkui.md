# HarmonyOS Command Prompt: /harmonyos-arkui

Use this prompt to run the `harmonyos-deveco` ArkUI-translation checkpoint.

## Arguments

## Delegation

- Produce an ArkUI Landing Contract: tokens, page structure, component split, state owners, routes, resources, permissions/config, and validation plan.
- Verify concrete HarmonyOS API/config facts with local docs search before asserting them.
