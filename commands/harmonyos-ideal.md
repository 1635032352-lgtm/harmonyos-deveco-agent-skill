# HarmonyOS Command Prompt: /harmonyos-ideal

Use this prompt to run the `harmonyos-deveco` ideal-capture checkpoint.

## Arguments

`$ARGUMENTS`

## Delegation

Apply the `harmonyos-deveco` skill.

- Run only `/harmonyos-ideal` unless the user explicitly asks to continue.
- Produce a Product Ideal Packet: promise, target user, scene, temperament, first value, non-goals, safety boundaries, and evidence target.
- Ask at most one product-shaping question when the answer would change the product shape, architecture, or acceptance evidence.
- Do not write ArkTS or edit DevEco files in this checkpoint.
